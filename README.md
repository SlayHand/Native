# Furniture App (Expo + Expo Router)

## Setup

1. Installi dependancies:

   ```bash
   npm i
   npx expo install expo-image-picker

   ```

2. Käivita dev server:
   npm run start

```
3. Käivita login server:
npm run dev

Andmed

Tooteid laaditakse: https://fakestoreapi.com/products

Kategooriad: https://fakestoreapi.com/products/categories

Navigeerimine

Avaleht: / (tabs/index)

Detail: /(app)/product/[id] (näiteks /(app)/product/1)

Märkused

“Contact Seller” avab mailiklioendi mailto: lingiga.

Kui hiljem on mitu pilti per toode, saab lisada karusselli komponendi images: string[] pealt.
```
